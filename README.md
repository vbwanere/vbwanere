## Mini AMR
<table>
<tr>
<td width="70%">
The team developed an AMR using an ESP32-S2 Wrover microprocessor controlling a two-wheel differential drive system with TT motors. The robot can perform neutral steering and demonstrate adaptive behaviors like auto-reverse when stuck. This has been achieved through IR-retroreflective sensors, the robot is powered by a dual power source (power bank for ESP32, 9V batteries for motors) with completely soldered circuits for robustness. The robot performed excellently across all tasks: achieving smooth wall following with seamless obstacle avoidance.
</td>
